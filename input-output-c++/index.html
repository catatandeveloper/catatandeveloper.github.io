Input Output C++

Dalam bahasa pemrograman C++, input dan output adalah dua operasi dasar yang sering digunakan untuk berinteraksi dengan pengguna atau perangkat lain. Operasi ini biasanya dilakukan dengan menggunakan standar library iostream yang menyediakan fungsionalitas untuk membaca data (input) dan menulis data (output).

Input dan Output

sederhananya input adalah masukan yang diterima dari terminal sehingga datanya bisa diolah di dalam kode, sedangkan output adalah data keluaran yang di tampilkan di terminal, berikut contoh input dan output:

#include <iostream>

int main()
{
    int number;
    std::cout << "masukan angka : ";
    std::cin >> number;

    std::cout << "angka yang anda masukan adalah : " << number << std::endl;
    return 0;
}

compile dan running programnya

$ g++ main.cpp
$ ./a.out

Penjelasan

  • cin: Digunakan untuk mengambil input dari pengguna. Pada contoh ini, cin digunakan untuk menerima input angka dan menyimpannya dalam variabel angka.
  • “»” : Operator ini digunakan untuk mengarahkan input dari cin ke variabel yang sesuai.
  • cout: Digunakan untuk menampilkan pesan kepada pengguna.

perlu kita ingat kembali ya “std::” adalah namespace dari standart library c++, jadi jika tidak menggunakan “using namespace std;” maka kamu harus memakai “std::” atau namespace setiap pemanggilan standart library.

Contoh lain

#include <iostream>
#include <string>
int main()
{
    std::string name;
    std::cout << "masukan nama anda : ";
    std::cin >> name;

    if(name == "dnan")
        std::cout << "Hallo : " << name << std::endl;
    return 0;
}

compile dan running programnya

$ g++ main.cpp
$ ./a.out
