Instalasi C++

C++ merupakan bahasa pemrograman yang multiplatform sehingga bisa di install di semua sistem operasi, seperti windows, linux, mac dan juga bsd. Berikut instalasi c++ pada windows dan linux.

Windows

MinGW adalah kompiler C/C++ asli (GCC) yang memiliki pustaka yang dapat didistribusikan secara gratis untuk membangun aplikasi diWindows. Dalam pembelajaran ini kita akan menggunakan MinGW sebagai compiler pemrograman kita.

  • unduh MingW pada link ini.
  • klik 2 kali untuk mengeksekusi MinGW.exe
  • klik install.
  • pilih penempatan folder, secara default di C:\MinGW kemudian continue
  • pilih MinGW Base System dan g++ compiler (bisa anda lihat di Description)
  • klik instalastion -> apply changes -> apply
  • tunggun proses sampai selesai
  • masukan bin dalam path windows dengan masuk ke environment.
  • pilih path dan edit.
  • klik edit dan masukan C:\MinGW\bin dan klik OK.

Linux

pada os linux instalasi g++ umumnya sudah terinstall secara default. tetapi jika anda mengetikan “g++” dan hasilnya “command not found” berarti c++ belum terinstal di komputer kalian.

install c++

$ sudo apt install g++
